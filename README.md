# Практическая работа №4

### Проект: Место.


В данной практической работе была сделана верстка адаптивного (респонсив) одностраничного сайта. Адаптивная верстка выполнялась исходя из макета в фигме под брэйкпоинты на 1280px (дэсктоп) и 320рх (телефон).

**Способ верстки - mobile first** (от мобильного телефона с разрешением экрана по ширине равной 320 пикселей и до экрана компьютера с разрешением 1280 пикселей и больше).

В лендинг была добавлена динамика с помощью JS. По клику на кнопку на странице появляется поп-ап с формой, где можно изменить строки с именем и деятельностью пользователя.


В ходе работы были применены:

· Взаимодействие с пользователем с помощью JS.

· Медиазапросы - построение адаптивной верстки при помощь брейкпоинтов.

· Построение внутренней структуры сайта по БЭМ-методологии (Nested).
